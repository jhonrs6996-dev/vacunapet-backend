Acerca de esta aplicación

Esta es una aplicación sencilla para llevar el control de mascotas. Permite registrar mascotas, ver su información, editarla y eliminarlas. También se pueden guardar vacunas, diagnósticos, recetas y prevenciones.

La idea principal es tener todo en un solo lugar, de forma ordenada y fácil de usar. El sistema calcula la edad de cada mascota automáticamente según su fecha de nacimiento.

¿Cómo funciona?

  • El usuario se registra e inicia sesión.
  • En el dashboard ve todas sus mascotas.
  • Puede agregar nuevas mascotas con nombre, especie, raza, foto y fecha de nacimiento.
  • Al ver una mascota, aparece su perfil completo con su información.
  • Se pueden agregar vacunas, diagnósticos, recetas y prevenciones.
  • Todo queda guardado en la base de datos.

Tecnologías utilizadas

  • Python + Flask
  • SQLite para guardar los datos
  • Bootstrap para el diseño
  • HTML para las pantallas

Este proyecto fue desarrollado con fines educativos, buscando crear algo funcional, simple y coherente con los conocimientos del estudiante.
